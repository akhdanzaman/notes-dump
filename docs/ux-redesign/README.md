# Notes Dump UX Redesign Roadmap

References used:
- Notes reference: https://dribbble.com/shots/26259906-Modern-Note-Taking-App-UI-Clean-Minimal-Mobile-Design
- Finance reference: https://dribbble.com/shots/25580105-Minimalist-Finance-Dashboard-UI

## Redesign order (one tab at a time)
1. Library (notes-first, closest to note-taking reference)
2. Money (closest to finance reference)
3. Summary (hybrid of notes + finance overview, but reduced to one hero + one feed)
4. Plan (task planner with minimal lane logic)
5. Calendar (calendar + agenda, low-noise)
6. Control Center (settings cleanup after main surfaces are aligned)

## Rules for this redesign
- One hero per screen
- One primary CTA per screen
- No duplicate metrics across header/body
- List-first over dashboard-first
- A card must earn its place; if a section repeats existing info, remove it
- Accent color only for actions, status, or priority
- Use whitespace and typography for hierarchy before adding containers

## Deliverable style
Each tab gets:
- Purpose
- Key user jobs
- Screen hierarchy
- Mobile wireframe
- Interaction rules
- What to remove from current UI
- Implementation checklist
